Sobre

Ninja Kombat é um jogo de batalha local para duas pessoas, jogado no mesmo computador.
O objetivo é vencer uma disputa melhor de 3, usando movimentação, disparos e estratégia para derrotar o adversário.

O projeto foi pensado para ter uma interface simples, mas ainda ser divertido.
Desenvolvido pelo grupo Dourado: Daniel Vinícius e Daniel Meireles.

Instruções para executar

Disclaimer: é essencial ter o Python 3.12 instalado na máquina.


  1. 1. Baixe e descompacte o arquivo .zip do jogo
  2. 2. Abra a pasta do projeto no PowerShell
  3. 3. Rode o script: .\setup.bat
  4. 4. Garanta que esteja usando o .venv: .venv\Scripts\activate
  5. 5. Execute o jogo: python3 src/jogo.py
