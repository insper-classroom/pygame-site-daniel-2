Descrição

Fey Gladiators é um jogo de batalha local para duas pessoas, jogado no mesmo computador. O objetivo é vencer uma disputa melhor de 3, usando movimentação, disparos e estratégia para derrotar o adversário.

O projeto foi pensado para ter uma interface simples, com telas bem separadas para início, instruções, seleção de personagem, combate e fim de jogo.

Instruções para executar

  1. Baixe e descompacte o arquivo do jogo.
  2. Abra a pasta do projeto no terminal.
  3. Instale as dependências, se necessário: pip install pygame.
  4. Execute o jogo com: python3 src/jogo.py.
