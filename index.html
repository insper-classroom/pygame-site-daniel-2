Sobre

Fey Gladiators é um jogo de batalha local para duas pessoas, jogado no mesmo computador.
O objetivo é vencer uma disputa melhor de 3, usando movimentação, disparos e estratégia para derrotar o adversário.

O projeto foi pensado para ter uma interface simples, mas ainda ser divertido.
Desenvolvido pelo grupo Dourado: Daniel Vinícius e Daniel Meirelles.

Instruções para executar

  1. 1. Baixe e descompacte o arquivo .zip do jogo
  2. 2. Abra a pasta do projeto no terminal
  3. 3. Instale as dependências: pip install pygame
  4. 4. Execute o jogo: python3 src/jogo.py
